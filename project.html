Análise de Dados com SQL

Utilizando um banco de dados no Excel, desenvolvi uma solução de filtragem SQL para analisar os dados de compras no Off2On. Neste projeto pude filtrar os dados por diversos critérios, como data, valor, região e vendedor, gerar relatórios personalizados e extrair informações relevantes, como o valor total de vendas por mês, os vendedores e filiais mais ativos e etc. A análise revelou o registro de um vendedor com um perfil destoante considerável dos demais, podendo trazer implicações significativas para detecção de fraudes e apontando a necessidade de investigação mais a fundo das causas desta anomalia.

Criação de Funções com SQL

Com o objetivo de otimizar a manipulação de dados em projetos SQL, desenvolvi um conjunto de funções personalizadas. Essas funções podem ser facilmente integradas a diversas aplicações, agilizando processos e garantindo a qualidade dos dados.
